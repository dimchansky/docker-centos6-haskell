Haskell for CentOS 6
--------------------

## Contents

This image ships a minimal Haskell toolchain with the following packages installed to
`/opt/{name}/{version}` and added to the `PATH`:

| package         | version    |
|-----------------|------------|
| `alex`          | `3.1.4`    |
| `cabal-install` | `1.22.2.0` |
| `happy`         | `1.19.5`   |
| `c2hs`          | `0.25.1`   |
| `ghc`           | `7.8.4`    |
